A Brand New Start

Mar 11, 2016

Sorry I have to say goodbye to Jekyll, thanks for the support from the whole community, I’ve been using Jekyll for almost three years, which not only taught me how to build static web pages, but also entertained me for writing blog like a hacker. I won’t become a front-end software developer if I hadn’t spent lots of nights customizing my personal blog.

The reason why I switched to Hexo is mainly I wanted to build the blog with Node.js related techniques. Hexo seems to be cool, open and most importantly there are some fantastic themes.

This blog is still the place where I’m gonna speak freely, I mean, you will see some blogs like 胡言乱语. Ignore them, lol.

Et voilà, allez, on recommence!
